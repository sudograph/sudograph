# Custom scalars

Custom scalars (scalars that you define) are not yet supported. You'll have to work with the included scalars:

* [Boolean](./schema-scalars.md#boolean)
* [Date](./schema-scalars.md#date)
* [Float](./schema-scalars.md#float)
* [ID](./schema-scalars.md#id)
* [Int](./schema-scalars.md#int)
* [String](./schema-scalars.md#string)
